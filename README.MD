# WP-SERVER
bash script for initial server setup

### Usage
Clone: 
```
git clone https://github.com/alisonbutcher/wp-server.git
```


```
cd wp-server
```

Check that server_name matches your domain name
```
nano website.com
```

Make script runnable
```
chmod +x lemp.sh
```

Execute script
```
./lemp.sh 1
```
